# **App Name**: NovaKaur

## Core Features:

- Intelligent Recruiter Pitch Tool: A generative AI tool that synthesizes Simran's ML and Full Stack skills into a personalized introduction message for potential employers.
- Dynamic Skills Matrix: An interactive, animated card-based interface for displaying proficiencies across Languages, AI/ML, Full Stack, and Databases.
- Project Showcase Hub: Comprehensive presentation of projects like KIZASHI and Pehchaan with integrated tech-stack badges and impact metrics.
- Vertical Education Timeline: A chronologically mapped layout for MCA, BCA, and high school academic history with integrated CGPA indicators.
- Certifications Vault: Horizontal glassmorphism scrolling tiles displaying verified credentials from Google, IBM, and Coursera.
- Unified Contact Portal: Centralized interface for LinkedIn, GitHub, and direct email communication, with localized Kolkata presence.
- Resume Cloud Access: Storage and download infrastructure for official documentation via a recommended Firebase backend.

## Style Guidelines:

- Primary Color: Deep Electric Violet (#9368e3), providing a high-contrast focal point that signals intelligence and engineering sophistication.
- Background: Midnight Obsidian (#120e16), a heavily desaturated deep violet that provides a premium, high-tech backdrop.
- Accent: Bright Cyan-Blue (#7599ef), used for active states, CTA buttons, and secondary highlights to ensure variety within an analogous scheme.
- Headline Font: 'Space Grotesk' for a computerized, tech-forward, and scientific aesthetic suitable for high-impact titles.
- Body Font: 'Inter' for a clean, neutral, and objective look, facilitating readability across project descriptions and CV details.
- A dark card-based design system featuring glassmorphism elements, soft outer glows, and a responsive vertical grid structure.
- Micro-interactions including floating profile photo effects, cursor-linked glow highlights, and subtle fade-ins using Framer Motion.
